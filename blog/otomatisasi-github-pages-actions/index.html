Otomatisasi GitHub Pages & Actions

Dengan GitHub Actions, Anda bisa mendeploy website Jekyll tanpa harus build lokal setiap kali ada perubahan.

Keuntungan

  • Deploy otomatis saat push ke branch utama.
  • Tidak perlu menjalankan build manual.
  • Lebih konsisten untuk tim.
